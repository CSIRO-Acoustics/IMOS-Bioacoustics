Acoustic Optic System geolocation

The AOS tab attempts to calculate the most likely position of a towed body from:
  • Depth of towed body
  • Depth below towed body
  • Bathymetry model

Usage

If the AOS tab is not present in dataview, use the File, Load scene menu to load AOS.scene. In the Filter SD tab use the Browse button to select the bathymetry grid to use (just typing in the grid name isn't sufficient). The following variables on the Filter SD tab are used in determining position. with the suggested settings:
Port Sector (deg) 5Maximum angle to port for possible locations
Starboard Sector (deg) 5Maximum angle to starboard for possible locations
Exclude lower range (%) 50Minimum percentage of range for possible locations
Exclude larger range (%) 130Maximum percentage of range for possible locations
Total depth error (m) 2Maximum difference between bathymetry and sum of depths
Wire:depth ratio 2Calculate wire out as this times depth of body. The Minimum check box should not be checked.
Auto-adjust wire:depth .01Amount to add/subtract to Wire:depth ratio each time mean possible location is significantly different to range
Smooth heading (fixes) 500Number of GPS fixes to use for calculating ship's heading
Max beacon speed (kn) 6Constrain body to a maximum speed through the water
Bathymetry offset (m) 0Systematic error in bathymetry/depth calculation
Winch (m) 0Added to range calculated from wire out
Current speed (m/s) 0Speed of current used to determine set
Current direction (deg) 0Direction of current
In the EchoView tab select the files containing the depth of platform and depth below platform information using the Browse button.

In the Replay settings dialog for the depth of platfrom file set a delay of 30 ms (higher to slow down the replay) select Synchronize and Pause all. [The program is multi-threaded with separate threads for displaying, calculating values and reading data and using calculated values. If there is insufficient delay in reading the background calculation thread may lag behind and stale values may be used for distance from ship.]

In the Replay settings dialog for the depth below platfrom file select Synchronize and Pause all (they should already be selected).

While Pause all is selected click the Start button for both files, then unselect Pause all to start the replay.

Show incoming data should be deselected for best performance.

During replay all the values, including the wire:depth ratio can be manually adjusted to improve performance.

When finished recording the track go to the Tracks tab and click the Save button for the Maximum range behind vessel track. Select the Echoview .gps.csv format, enter a filename (you can use the Browse button) then click Save. If you do multiple runs of the replay you should Reset the track before the final run.
