Underway

Underway data is data logged by the ship's system while the vessel is underway.

Data is logged from the following instruments:
GPSThe Geographic Positioning System provides information on the location of the vessel
mode
pdop
speed_ogSpeed Over Ground - the speed of the vessel relative to the earth, as calculated by the GPS
speed_owSpeed Over Water
course_ogCourse Over Ground - the direction the vessel has moved, relative to the earth.
headingThe direction the ship is pointing
EKThe Simrad EK500 echosounder
sea_depthThe depth of the sea floor, according to the current settings of the echosounder. The accuracy of this measure will depend on the settings and whether they match the current environment. In particular this measure is sensitive to the speed of sound through water, which varies with water temperature and conductivity.
Data TakerThe data taker connected to the meteorological sensors
air_tempAir temperature
water_tempSea water temperature
wind_speedWind speed (knots?)
wind_dirWind direction in degrees true
light
ThermosalinographWater characteristics as measured at the thermosalinograph
salinity
water_tempSea water temperature at the thermosalinograph. The water temperature is likely to have been affected by travel through the ship to the instrument.
conductivity
fluorescence

Loading

Data can be loaded for only one area at a time, although more than one attribute of the dataset may be displayed (using different symbols for each attribute). Once a dataset is loaded the geographic bounds and time range options are disabled and ignored. Removing all loaded datasets will enable new bounds to be entered.

You may restrict the data loaded by either geographic bounds or time range, or both. At least one restriction should be applied as this is a very large data set.

Attribute

Which data attribute you wish to display. This is the attribute to be used for colouring the data points. You must select one.

All attributes may be viewed in the data point details panel for underway data.

More than one data attribute may be displayed by selecting a different attribute and symbol (and optionally colour table) and loading it. Each attribute and symbol must be selected one pair at a time, and the Load button pressed. Loading an additional attribute of an already loaded dataset is does not take anything like as long as loading the dataset.

Only one data set can be loaded at a time, even though multiple attributes of the dataset may be plotted.

Symbol

Select which symbol you would like to use to display this attribute. You must select one.

Geographic bounds

You may restrict the data loaded to a geographic area. Fill in the bounds either by typing them in or by middle dragging the mouse to define the area. Double middle click of the mouse will clear the current bounds.

Once a data set has been loaded the geographic bounds will be ignored until the data set has been removed. A data set can be removed either by the RemoveAll button or by the File, Clear menu option.

Time range

You may restrict the data loaded to a time range. Fill in the start and end times for data to load. All times are in UTC (GMT).

Colour table

You must select a colour table to use from the predefined list.

You may choose to apply either a linear or logarithmic scale to colour the data. Some data sets, such as single beam indices and echosounder depth are better displayed using a log scale.

The colour table may either range over the set of data values, "Auto range", of range over the minimum and maximum values you enter. You can reverse the colour scale by setting the minimum value greater than the maximum value.

Load

The Load button will load all operations of the currently selected types which fall within the region in to the display area.

Cancel

The Cancel button will be available between when you press the Load button and when the data is displayed. Pressing the Cancel button will abort the load.

The data may take some time to load, especially if large or no bounds are applied. While the data is loading the program may still be used as normal, albeit with reduced performance as the data loading activity occurs in the background.

Remove

The Remove button will remove the selected attribute from the display.

Remove All

The RemoveAll button will remove the current underway data set, and allow a new data set to be selected.

Re-colour

The Re-colour button will apply the currently selected colour table and range to the currently selected data set.
