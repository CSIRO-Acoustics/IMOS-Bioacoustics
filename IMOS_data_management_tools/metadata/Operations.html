Operations

The operations data set is the list of ship's operations. These activities are recorded into the on ship database as the activity occurs.

All times are in UTC (GMT). The local timezone offset is recorded.

All positions are ship positions.

For each operation the time and location is recorded for each of:
Deployment startWhen the equipment was put over the side
Operation startWhen sampling commenced
Operation endWhen sampling finished
Deployment endWhen the equipment came on board
Operations are displayed as a straight line between operation start and operation end locations with a symbol representing the operation type at each end of the line. The line does not represent the ship's track during that period. The operation start and end locations represent the ship's location, not the location of the sampling instrument, which in the case of towed equipment, such as sleds and video cameras, may have a layback in the order of twice the water depth.

Loading

Types

You may restrict the operation types to those you select from the list. If no types are selected they will all be loaded.

Geographic bounds

You may restrict the operations to those that include a particular area. Enter the north, south, east and west latitudes and longitudes. Any operations which are within or pass through the region you defined will be loaded. An operation "passes through" the region if the rectangle defined by the operation start and end locations overlaps with the region at all.

Load

The Load button will load all operations of the currently selected types which fall within the region in to the display area.

Cancel

The Cancel button will be available between when you press the Load button and when the operations are displayed. Pressing the Cancel button will abort the load, but will not remove any already loaded data. It will normally take very little time to load the operations data, so the Cancel button may not be available.

Remove

The Remove button will remove those operations of the type you currently have selected.

Remove All

The RemoveAll button will remove all operations from the display.
